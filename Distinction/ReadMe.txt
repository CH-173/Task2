Сценарий: Вывод количества уникальных чисел в массиве
Дано: Программа просит сначала ввести <количество элементов массива> а затем <целочисленные элементы массива> в порядке возрастания
Когда: Пользователь вводит целочисленное <количество элементов массива> а затем <целочисленные элементы массива> в порядке возрастания
То: Программа выводит сообщение "В массиве <количество> уникальных элементов."

Примеры:

|количество элементов массива|
|1                           |
|2                           |
|...                         |


|целочисленные элементы массива|
|-1                            |
|0                             |
|1                             |
|2                             |
|...                           |

|количество|
|1         |
|2         |
|...       |


----------------------------------------------------------------------

Сценарий: Ошибка (отрицательное количество входных элементов)
Дано: Программа просит сначала ввести количество элементов массива а затем целочисленные элементы массива в порядке возрастания
Когда: Пользователь вводит отрицательное целочисленное <количество элементов массива>
То: Программа выводит сообщение "Количество элементов массива не может быть отрицательным числом!"

Примеры:

|количество элементов массива|
|-1                          |
|-2                          |
|...                         |

Сценарий: Ошибка (нулевое количество входных элементов)
Дано: Программа просит сначала ввести количество элементов массива а затем целочисленные элементы массива в порядке возрастания
Когда: Пользователь вводит нулевое (0) количество элементов массива
То: Программа выводит сообщение "Количество элементов массива не может равняться нулю!"


-----------------------------------------------------------------------

Сценарий: Ошибка (элементы массива введены не в порядке возрастания)
Дано: Программа просит сначала ввести количество элементов массива а затем целочисленные элементы массива в порядке возрастания
Когда: Пользователь вводит ненулевое целочисленное количество элементов массива а затем целочисленные элементы массива не в порядке возрастания
То: Программа выводит сообщение "Вы ввели элементы не в порядке возрастания!"


-----------------------------------------------------------------------

Сценарий: Ошибка (не целочисленные входные данные)
Дано: Программа просит сначала ввести количество элементов массива а затем целочисленные элементы массива в порядке возрастания
Когда: Пользователь вводит не целое число (либо не являющееся числом) количество элементов массива и/или не целочисленые (или не являющиеся числами) элементы массива
То: Программа выводит сообщение "Вы ввели не целое число элементов массива либо не целочисленный элемент массива!"
